#Proyecto#
1- Cabecera de nuestra página: Crear una sección en HTML con un párrafo y una cita, aplicando propiedades de texto en CSS.
2- Propiedades de Texto en CSS: Aplicar propiedades de texto como tamaño, alineación, transformación y decoración a párrafos y citas.
3- Propiedades de Caja en CSS: Diseñar dos divs con propiedades de caja, como padding, margen, borde, y sombra.
4- Posición Relativa y Absoluta en CSS: Implementar cajas posicionadas con position: relative y position: absolute para mostrar distintas ubicaciones.
5- Media Queries en CSS: Ajustar el diseño para pantallas pequeñas utilizando una media query, cambiando colores y tamaños de fuente.
6- Principios de Grid en CSS: Crear una cuadrícula con CSS Grid, definiendo columnas adaptables y estilizando los elementos dentro del grid.
7- Principios de Animaciones en CSS: Añadir animaciones y transiciones a elementos div utilizando @keyframes y efectos de hover.
